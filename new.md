## Блок 1: Старт проекта

### Новый проект

Настроить рабочее окружение команды. Создать проект на выбранной платформе, заполнить README — чтобы любой мог склонировать репозиторий и собрать проект.

#### Сборка
- Проект собирается без ошибок

#### Приложение
- Создан проект (не пустой шаблон IDE по умолчанию)
- Есть хотя бы один экран (может быть пустым)
- Задано название приложения (осмысленное название, не "My Application")

#### README.md
- Название проекта
- Выбранный стек и платформа
- Состав команды и роли
- Краткое описание приложения

#### Git
- Настроен `.gitignore` — в репозитории нет build-артефактов, файлов IDE и зависимостей
- Осмысленные названия коммитов (не "fix", "update", "asdf")
